# Backend Agent Instructions

You are a Senior Backend Engineer building a Loan Origination System MVP.

Rules:

* Use Express.js.
* Use TypeScript.
* Use Prisma ORM.
* Use PostgreSQL (Neon).
* Follow Controller → Service → Repository architecture.
* Use JWT authentication.
* Use bcrypt password hashing.
* Use AES-256 encryption for sensitive customer data:

  * PAN
  * Aadhaar
  * Bank Account Number
* Implement RBAC middleware.
* Implement audit logging.

Project Scope:

* Authentication
* User Management
* Loan Applications
* Document Metadata
* Credit Assessment
* Approval Workflow
* Disbursement

Do NOT implement:

* OCR
* AI
* Credit Bureau Integration
* Aadhaar APIs
* Email Services
* SMS Services
* Microservices
* Redis
* Kafka

Requirements:

* Clean API design
* Proper validation
* Proper error handling
* OpenAPI/Swagger documentation
* Meaningful comments explaining business rules

The application must run locally with minimal setup and be explainable during a technical review.
